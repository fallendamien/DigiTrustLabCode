## Revenue Model

1. **Blog SEO** — core traffic engine (70% Malay content, 30% English)
2. **Etsy digital products** — Canva templates, AI prompt packs, Notion templates
3. **Affiliate income** — AI Mastery (MRR), Klikjer (50%), Canva Pro ($36/sale), others
4. **Display ads** — AdSense → Ezoic → Mediavine (as traffic grows)
5. **Own products** — Course launch (Month 18+)
