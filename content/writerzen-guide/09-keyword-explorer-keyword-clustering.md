# 2.7 Keyword Clustering in Keyword Explorer

## Overview

This lesson covers the Keyword Clustering function in WriterZen — how it works, how to set it up, and how to use the results to build a powerful content plan.

## What Is Keyword Clustering?

Keyword clustering is a technique that identifies the main themes or topics that a group of keywords belongs to, then organises those keywords into clusters based on those themes. This enables you to optimise a single page for multiple keywords rather than just one.

**Example:** "good web designer in London" and "great web designer in London" are similar in meaning and context. A single page can likely rank for both — so they belong in the same cluster.

Key benefits of keyword clustering:
- Reduces the number of articles you need to write
- Enables each article to rank for multiple keywords
- Builds topical authority by turning clusters into in-depth articles covering a topic fully

> **Note:** All keyword data is sourced from Google's proprietary API — no guesswork involved.

## How to Use Keyword Clustering

**From Keyword Explorer:**
1. Scroll down to the Keyword Data tab
2. Click the **Cluster** button to run the clustering function

**From an external keyword list:**
1. Head to the **Keyword Importer** tool
2. Import your keyword list
3. Click the **Cluster** button

## Creating a Cluster Project

The quality of your clusters depends on the quality of your input keyword list. Before clustering, use the **Golden Filter** to identify keywords with the highest ranking potential.

**Example filter settings for a low-authority site:**
- Golden Score: max 10
- Search Volume: min 100
- All-in-Title: max 10

**Steps to create clusters:**
1. Apply your filters and select the filtered keyword list
2. Click **Cluster** → choose **Cluster Selected**
3. Set the **clustering level** (default is 3 — meaning each keyword in a cluster must appear together in at least 3 SERP results)
4. WriterZen checks the top 10 SERP positions for every keyword and groups them based on the similarity of their search results
5. Name your cluster project and click **Activate**
6. You and your team are notified via in-app notification and email when clustering is complete
7. Results are saved in the **Keyword Importer**

## Accessing Your Clusters

Completed clusters are found in the **Keyword Planner**. If you ran clustering from Keyword Explorer, you will be taken to the results in Keyword Planner automatically.

**Two views available:**

| View | What It Shows |
|---|---|
| **Cluster View** (default) | Organised left to right — Topics at the highest level, Clusters (subtopics) within each topic |
| **All Keyword View** | Flat list of all keywords across all clusters |

Think of **Topics** as parent themes and **Clusters** as subtopics within each theme.

Each cluster contains insights including:
- Keyword metrics
- Search intent classification
- Position in the buying journey

These insights are covered in detail in the Keyword Planner lessons.

## Impact on Your Content Plan

Keyword clusters directly shape your content plan — they give you a big picture of how to create, optimise, and organise content on your website.

**Old approach (pre-clustering):** 100 keywords = 100 articles
**With clustering:** Related keywords are grouped together → fewer articles needed, each ranking for multiple keywords

The Keyword Planner lessons will cover:
- How to transition from clustered keywords to finished articles that rank
- How to use WriterZen's keyword insights to pick the right keywords for each article
- How to generate a detailed content brief for each cluster
- How to use that brief to create high-quality articles quickly

## Interlinking Articles

Properly interlinking your articles creates a powerful authority structure:

- All subtopic articles strengthen the page authority of the pillar page
- The pillar page passes authority back to all subtopics
- Closely related topics, properly interlinked, build the overall topical authority of the entire website

## Boosting Topical Authority on Existing Sites

Keyword clusters can also be mapped to existing pages rather than new articles. Use clusters to:
- Add new content sections to existing articles
- Re-optimise header tags, body text, and internal links
- Fill in content gaps for topics you previously missed

**Example:** A website covering Toyota cars that took a break from publishing can come back, identify missed models using clustering, and restore and boost its topical authority by covering those gaps.

## Key Takeaway

> *"Search is not about what people find — it's about how people think."*

Keyword clustering reflects how real users think and search. There is no single right way to map out topics — content is always dynamic, so there is plenty of room to get creative. Use clustering as a foundation, then build your strategy around it.
