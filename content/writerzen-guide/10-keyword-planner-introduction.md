# 3.1 Introduction to Keyword Planner

## Overview

The Keyword Planner bridges the gap between Keyword Explorer and Content Creator — completing the full content workflow inside WriterZen.

## What You Can Do with Keyword Planner

- Import keywords from both internal and external sources and group them automatically into topics and clusters
- Optimise keyword planning with advanced qualitative insights such as keyword intent, buying journey stage, and SERP type
- Get a suggested content brief — the Keyword Planner automatically transforms keyword insights into actionable steps
- Simplify the content production process with a user-friendly interface and AI support

## How It Works (Mechanism)

**Data Crawling**
WriterZen crawls Google's search results, Google Keyword Planner, and the Google Keyword Suggest database to find all relevant data related to your input keywords.

**Auto-Clustering**
All keywords are automatically clustered into topic groups — showing you which keywords belong together for easy rankings.

**Insights from Keywords and Clusters**
WriterZen's built-in language model and OpenAI's algorithms simplify and categorise qualitative insights from crawled keywords, based on the similarity of the SERPs. This goes beyond numbers to surface intent-level understanding.

## Setting Up a New Project

There are two ways to create a Keyword Planner project:

### Option 1: From Keyword Explorer (Auto)
When you run the Clustering function inside Keyword Explorer, you are automatically taken to the Keyword Planner when results are ready. The project is created automatically and the seed keyword becomes the project name.

### Option 2: Import Keywords Directly

1. Access Keyword Planner via the navigation menu
2. Click **Import Keywords**
3. Name your project, choose a location, and select the target language
4. Check **Save language and location** to set these as your default
5. Import your keywords — three options available:
   - Type keywords directly into the text area
   - Upload your own text file
   - Import any keyword list created inside WriterZen
6. Set your **clustering level** (default: 3 — each keyword in a cluster must appear in at least 3 SERP results)
7. Toggle **Golden Filter** on if you want access to WriterZen's Golden Filter after clustering — useful for re-optimising your keyword list later
8. Click **Add** to start the clustering process

> **Import limit:** Depends on your plan — the highest tier allows up to 20,000 keywords. Each line = one keyword, including keyword phrases.

You and your team are notified via in-app notification and email when clustering is complete.

## How Clustering Works (Recap)

WriterZen checks the top 10 SERP positions for every keyword in the list and groups them based on the similarity of their search results.

With a clustering level of 3: when two keywords share three or more similar results in the top SERPs, they are grouped into the same cluster.
