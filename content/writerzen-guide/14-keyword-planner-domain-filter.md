# 3.5 Domain Filter in Keyword Planner

## Overview

The Domain Filter is a powerful competitive analysis tool inside Keyword Planner. It gives you a bird's eye view of all domains ranking for your keyword list and enables you to find hidden opportunities by analysing what competitors are — and are not — ranking for.

---

## Overview Tab (Default)

The Overview tab shows all domains ranking for your keyword list, from left to right:

| Column | What It Shows |
|---|---|
| **Domain** | Complete list of domains ranking for your keyword list |
| **DA (Domain Authority)** | Authority level of each competing domain |
| **Volume %** | Market share — percentage of total search volume each domain captures |
| **Keywords** | Number of keywords from your list each domain ranks for |
| **URLs** | Number of URLs each competitor uses to rank for that keyword list |
| **Weighted Rank** | Average ranking of each domain's keywords, weighted by search volume |

### What Is Weighted Rank?

A standard average rank treats all keywords equally. Weighted Rank accounts for search volume — giving more importance to higher-volume keywords.

**Example:**
- Keyword A: rank 2, volume 1,000
- Keyword B: rank 3, volume 200
- Keyword C: rank 6, volume 30,000

Standard average = 3.67. But Weighted Rank multiplies each keyword's rank by its search volume, then divides by total search volume — making rank 6 for keyword C (30k searches) far more significant than rank 2 for keyword A (1k searches).

> Weighted Rank is a more accurate measure of a domain's real-world competitive performance.

---

## Filter Rules Tab

The Filter Rules tab lets you identify which keywords a domain **is** or **is not** ranking for — revealing hidden opportunities and competitor weaknesses.

### Setting Up a Filter

1. Click **Add Filter**
2. Choose filter type:
   - Include keywords / Exclude keywords
   - Include clusters / Exclude clusters
3. Click the **Domain box** and select one or more domains
4. Set **Ranked** or **Unranked**
5. If Ranked: choose **Exact position**, **Range** (e.g. 1–3), or **Anywhere**
6. Click **Apply** → **Apply Changes**

You can combine multiple filter rules in one setting for advanced analysis.

## Use Cases

### Use Case 1: Market Overview — Is the Playing Field Even?

Open the Domain Filter Overview tab to instantly see:
- Which domains dominate your keyword list
- Each domain's market share (volume %)
- Whether one player controls the space or competition is spread out
- Set results to Top 15 to check if social media sites (Pinterest, YouTube) are capturing traffic

> **Example:** Pinterest at position 6 (12.6% traffic) and YouTube at position 7 (11.4%) = 24% combined traffic from user-generated content — easy to beat with a high-authority, well-optimised article.

---

### Use Case 2: Analyse Your Own Website Performance

1. In Filter Rules, check your own domain name
2. Set filter to **Include Keywords** → ranking positions **5–10**
3. Apply → see all keywords you currently rank for that need improvement
4. For each underperforming keyword, choose:
   - **Rewrite the existing article** and go deeper into the topic
   - **Create a new, better-optimised article** and use internal linking from the current ranking page to the new one

You can also filter for positions **2–5** to find keywords close to top 3 with room for improvement, or filter for **Unranked** to see all keywords you are not yet targeting.

---

### Use Case 3: Reverse Engineer Competitor Content Strategy

1. In Filter Rules → **Include Keywords**
2. Add your top 3 competitors
3. Set filter to: **Ranked** → **Top 10**
4. Apply → see exactly which keywords your competitors rank for
5. Check the intent column — if mostly informational, that tells you the content type you need to create
6. Switch to **Unranked** → see keywords your competitors are **ignoring** — these are your golden opportunities

> Keywords that strong competitors are not ranking for = lower competition = faster ranking potential for you.

---

### Use Case 4: Find Competitor Weak Spots

1. Filter Rules → **Exclude Clusters**
2. Select a dominant competitor (e.g. eatingwell.com)
3. Set to: ranking in **Top 5**
4. Apply → see clusters that competitor is NOT ranking well for

When you find a cluster where:
- The dominant competitor is absent from top 10
- Pinterest or YouTube is ranking (user-generated content)
- Low DA domains (e.g. DA 25) are in top 10

→ This is a high-value opportunity. Create a well-optimised article and you can expect to rank.

> **Super Tip:** When you find a keyword opportunity, always check the other keywords in the same cluster — they often have similar opportunities. Writing one pillar article can capture the combined search volume of the entire cluster.

---

### Use Case 5: Find Social Media Ranking Opportunities

1. In Overview Tab, set results to Top 15
2. Identify Pinterest and YouTube in the rankings
3. In Filter Rules → **Include Keywords** → add Pinterest + YouTube
4. Set to: ranking in **Top 10**
5. Apply → see the specific keywords these social sites rank for

For each keyword found:
- Check DA of other ranking domains in the SERP overview
- If low DA domains and/or a PDF are ranking in top 10, that is a clear signal you can outrank them

To refine further: add a second rule to **Exclude Keywords** your own domain already ranks for in top 10 → leaves only the uncontested opportunities.

## Summary: What the Domain Filter Enables

| Goal | Filter Setup |
|---|---|
| Market overview | Overview tab → set results to top 10–15 |
| Check own performance gaps | Include keywords → own domain → positions 5–10 |
| Find competitor weak spots | Exclude clusters → competitor → top 5 |
| Reverse engineer competitors | Include keywords → competitors → ranked top 10 |
| Find what competitors ignore | Include keywords → competitors → unranked |
| Beat social media traffic | Include keywords → Pinterest/YouTube → top 10 |
| Combine multiple rules | Add multiple filters in one filter set |

The Domain Filter is one of the most powerful tools in WriterZen's Keyword Planner — use it to map the competitive landscape, find gaps, and prioritise your content strategy with data rather than guesswork.
