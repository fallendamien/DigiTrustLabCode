## Chapter 1: Rename Topic / Cluster

You can rename any topic or cluster directly inside the Keyword Planner interface.

- Click on the topic or cluster name to edit it
- Renaming helps you organise your project with meaningful, custom labels rather than the auto-generated names from keyword data
- The default cluster name is taken from the keyword with the highest search volume in that cluster — renaming lets you override this with something more descriptive for your content plan
