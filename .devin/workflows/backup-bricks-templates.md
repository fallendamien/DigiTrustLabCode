# Backup Bricks Templates

Quick slash command to trigger a Bricks template backup.

→ **Load skill:** `.devin/skills/bricks-backup/SKILL.md`

## Usage

```
/backup-bricks-templates
```

Optional arguments:
- `all` — backup all 4 templates (default)
- `header` — backup only header (ID 21)
- `footer` — backup only footer (ID 46)
- `single` — backup only single post (ID 10)
- `archive` — backup only blog archive (ID 52)
